---

### 企画した内容

ホテルといった、宿泊業としては、空室がないほうが、利益です。

泊まるお客様としては、１泊でも長く予約する方が、安く予約できます。

この点を着眼し、退室するお客様と、入室するお客様の日付が、一致する場合、その予約を一つと予約とし、

利用者としては安く利用でき、業者としは空室がないようにするサービスを設けました。

#### 例

1. A顧客の場合、5月1日〜5月3日　：　２泊3日

2. B顧客の場合、5月3日〜5月5日　：　２泊3日

   ➡️４泊5日の予約として扱う。
